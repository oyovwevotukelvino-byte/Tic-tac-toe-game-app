🎮 Tic-Tac-Toe Game App

A fun and interactive Tic-Tac-Toe game built with React. This project demonstrates clean UI design, efficient game logic, and a nostalgic gameplay experience.
✨ Features

    Classic 3x3 grid gameplay

    Two-player mode (play with a friend)

    Computer AI mode (challenge the computer)

    Win, lose, and draw detection

    Responsive design for desktop and mobile

🚀 Tech Stack

    React (frontend framework)

    JavaScript (ES6+) for game logic

    CSS for styling

    Vite for fast development and build

🎯 Purpose

This project was created to:

    Practice React fundamentals

    Showcase clean code and UI design

    Provide a quick, nostalgic game experience
